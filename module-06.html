Module 06 • Build sequence

🏗️ Weekly Schedule Build

Build a week by preparing source documents first, running late shifts, running the weekly schedule, and resolving Staff Down gaps before publication review.

WhatWhyWhat to useHow to use itSelf-check
Module progress:

Learning objectives

  • List what to print or review before building the weekly schedule.
  • Explain why the build does not begin by running the schedule.
  • Run late shifts before the weekly schedule run.
  • Use Staff Down to identify and fill missing AM, PM, or full-day assignments.

Why this matters

  • The weekly schedule is built from constraints: days off, evenings, tumor boards, admin, seating, home requests, specialty needs, weekend context, and fairness grids.
  • Running the schedule before reconciling constraints can create conflicts that are harder to see and harder to correct later.

What to use

  • 1 - Evening Stats and Evening STAT Grid.
  • Schedule Check Grid and Schedule Check Grid Modalities.
  • Rad Weekly Evening Schedule and Weekly Rad QGenda Sheet.
  • Neuro Rads Fluoro Weekly Schedule.
  • Weekly Meeting Sheet, MILV Event Calendar, Shared Calendar, and SharePoint Tumor Board resources.
  • Weekend schedule for the weekend before and after the week being built.
  • QGenda vacation/time-off report for the current week and following week.
  • QGenda Make Schedule and Staff Down view.

How to use it

  1. Print or open all weekly-build resources before running the schedule.
  2. Set QGenda to the week being built. The build week runs Monday through Sunday; the vacation report is pulled Monday through Friday.
  3. X off unavailable blocks from vacation/time-off documents and verify them against QGenda and pending requests.
  4. Place IR radiologist locations on the Schedule Check Grid so every IR radiologist is seated somewhere for the full week.
  5. Add Home and Admin assignments in the grid and in QGenda.
  6. Mark fixed locations and always-evening radiologists.
  7. Add meeting and tumor board notes in QGenda when not already placed.
  8. Check weekly seating needs and assign required sitting coverage.
  9. Run late shifts first: Make Schedule > M-F > Task Run Filter: Run Late Shifts.
  10. Run the weekly schedule: Make Schedule > M-F > Task Run Filter: Run-Weekly Schedule.
  11. Use Staff Down: Admin > Display > Grid: Staff. Mark missing AM, PM, or all-day assignments.
  12. Fill remaining assignments manually, checking specialty, seating, fluoro, part-time restrictions, home/admin, and fairness grids.
  13. Prepare the schedule for leadership review before publication.
