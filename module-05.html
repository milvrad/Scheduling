Module 05 • Hands-on QGenda

✅ QGenda Core Procedures

Use QGenda consistently for task entry, notes, vacation days, reports, profile setup, and swap tracking.

WhatWhyWhat to useHow to use itSelf-check
Module progress:

Learning objectives

  • Enter a single vacation day and copy it to additional days correctly.
  • Run a QGenda vacation/time-off report for schedule building.
  • Apply task-entry rules for checked circles, PUB, Fluoro Outpatient tasks, and notes.
  • Understand why Track/Untrack is required after a swap.

Why this matters

  • QGenda is the live operational schedule. If QGenda is wrong, the schedule is wrong even if an email or worksheet says something else.
  • Consistent notes and tracking keep publishing, reconciliation, payroll, and reporting accurate.

What to use

  • Schedule tab for task entry, filters, three-dot task menu, notes, and swaps.
  • Stats tab and Vacation Tally for reviewing submitted requests.
  • Reports tool for Grid by Task, Excel format, and Pending Requests visibility.
  • Admin tab for Track/Untrack after swaps.
  • Settings > Staff for new staff profile setup when that role responsibility applies.

How to use it

  1. Use the checked circle option when adding tasks; use PUB when the task is confirmed and the workflow calls for it.
  2. Add a Fluoro Outpatient Assignment Task for every Fluoro assignment.
  3. Use notes for home/remote status, no-evening requests, special coverage, seating, request date, and location details.
  4. For a single vacation day, save the request, enter Vacation (Day), add the request-date note, submit, then copy the entry to remaining requested single days.
  5. Open Stats and the applicable Vacation Tally to verify the radiologist’s submitted requests.
  6. For schedule-building reports, choose Report Type: Grid by Task, Format: Excel, Date Range: Monday-Friday, and include Pending Requests when needed.
  7. When a swap is completed, go to Admin, filter both radiologists, Track the receiving radiologist, Untrack the original radiologist, and verify both changes.

Enter single vacation days

StepAction
1Print or save the email/request documentation before entering the schedule.
2Start on the first requested day and enter Vacation (Day).
3Add a note with the request date, such as Requested on 05.18.26.
4Click Submit.
5Use the three-dot menu on the right and choose Copy.
6Paste the copied vacation entry into each remaining single day requested.
7Open Stats and choose the correct period, such as Summer 2026.
8Scroll to the radiologist and snip/copy the list of submitted requests.
9Reply to the radiologist confirming the request has been noted and include the snip/list when appropriate.

Complete a swap and update tracking

StepAction
1Go to Schedule and add the radiologists involved to Filters.
2Find the assignment to move and click the three-dot menu.
3Choose Swap 1-Way. Select one assignment or multiple assignments as needed.
4Highlight all tasks to swap; selected tasks turn yellow.
5Click Done, choose the receiving radiologist, and click Add to Swapping Cart.
6Review the cart and click Submit Swaps.
7Go to Admin and filter to both radiologists.
8For the receiving radiologist, select the assignment, click Track, and Submit.
9For the original radiologist, select the assignment, click Untrack, and Submit.
10Verify the schedule display, receiving tracking, original untracking, and any notes before exiting.

Create a new staff profile in QGenda

StepAction
1Log into QGenda and open Settings from the top-right of the screen.
2Click Staff on the left options list.
3Best practice: copy a similar staff member profile.
4In the Copy dialog, choose To New Staff.
5Enter first and last name, abbreviation, modality text color, and personal email if required by current setup standards.
6Set sort order carefully so the radiologist falls alphabetically by last name.
7Check the applicable configuration selections from the reference profile.
8Click Copy to New Staff.
9Complete General and Contact Information as much as possible. Include phone, ASCOM, or reliable contact method if available.
10Configure skill set/schedule using confirmed information; update as additional details become available.
